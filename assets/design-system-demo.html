Design System · 设计系统

Meet your thinking partner

这是一套从 Claude 官网提取的完整设计系统，包含设计令牌、组件样式与代码片段。你可以把整页源码或其中的 CSS 变量直接复用到任何项目里。

01 · Colors

颜色 系统

基底是暖米色 #F0EEE6，文字偏黑棕，强调色是 Claude 标志性的珊瑚橙 #CC785C。整体氛围温和、纸质感，不刺眼。

03 · Buttons

按钮 组件

五种按钮形态覆盖了官网所有按钮场景：主按钮、强调按钮、描边、幽灵、胶囊标签。

主按钮 (Primary) — 深色，用于核心 CTA 如 "Try Claude"

强调按钮 (Accent) — 珊瑚橙，用于次级强调如 "Ask Claude"

描边按钮 (Outline) — 用于次级动作

幽灵按钮 (Ghost) — 用于导航和工具栏

胶囊标签 (Pill) — 用于快捷功能/分类标签

08 · Code

复制即用：CSS 变量

把下面这段贴进任何项目的根样式表，就拿到全套设计令牌。

/* === Claude Design Tokens === */ :root { /* 背景 */ --bg-primary: #F0EEE6; --bg-secondary: #EBE9DF; --bg-card: #FAF9F5; --bg-dark: #262624; /* 文字 */ --text-primary: #1F1E1B; --text-secondary: #5C5A53; --text-muted: #8E8A82; /* 品牌色 */ --accent: #CC785C; --accent-hover: #B86848; /* 字体 */ --font-serif: 'Fraunces', Georgia, serif; --font-sans: 'DM Sans', system-ui, sans-serif; /* 圆角 */ --radius-sm: 6px; --radius-md: 10px; --radius-lg: 16px; --radius-pill: 999px; } /* === 主按钮 === */ .btn-primary { background: var(--text-primary); color: var(--bg-primary); padding: 0.65rem 1.25rem; border-radius: var(--radius-sm); font-weight: 500; } /* === 强调按钮（珊瑚橙） === */ .btn-accent { background: var(--accent); color: #fff; } /* === 标题斜体强调（Claude 标志性） === */ h1, h2, h3 { font-family: var(--font-serif); font-weight: 400; letter-spacing: -0.02em; } .italic-accent { font-style: italic; font-weight: 300; }
